Analysis
========

.. raw:: latex

    \label{Analysis}

In der Analysis werden Funktionen und ihre Eigenschaften untersucht. Funktionen,
also eindeutige :ref:`Abbildungen <Abbildungen-Funktionen>`, weisen in
eindeutiger Weise eine Größe einer anderen Größe zu. Im gleichen Sinn werden
eindeutige Zuordnungen zwischen zwei (oder mehreren) Größen auch als
"funktionale Zusammenhänge" bezeichnet.

Zur Untersuchung von Funktionen zählen insbesondere das Steigungs- und
Krümmungsverhalten von Funktionsgraphen, ihr Verhalten im Unendlichen sowie die
Bestimmung von Flächeninhalten zwischen verschiedenen Funktionsgraphen.
